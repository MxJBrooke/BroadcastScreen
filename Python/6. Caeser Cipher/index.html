Caesar Cipher (Encryption)

Build Your Own Cipher Tool 🔐

A Caesar Cipher shifts letters in the alphabet.

Example: A → D (shift of 3)

Your program should include:

  • At least: 1 loop
  • At least: 2 if statements
  • User input
  • Output message

Step-by-step guide:

  1. Ask the user for a message
  2. Ask for a shift number
  3. Loop through each letter
  4. Shift each letter
  5. Print the result

Starter Code:

Scaffolded Code:

print("Caesar Cipher")

# Step 1: Get message
message = input("Enter your message: ")

# Step 2: Get shift
shift = int(input("Enter shift: "))

# Step 3: Store result
result = ""

# Step 4: Loop through message
for letter in message:

    # Step 5: Check if it's a letter
    if letter.isalpha():

        # Step 6: Convert to number
        number = ord(letter)

        # Step 7: Shift it
        number = number + shift

        # Step 8: Convert back
        new_letter = chr(number)

        result = result + new_letter

    else:
        result = result + letter

# Step 9: Output
print(result)
