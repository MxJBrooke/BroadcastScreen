Denary to Hexadecimal Conversion Coding Challenge

Denary to Hexadecimal Cheat Sheet (2 Digits)

Hexadecimal uses the digits 0–9 and the letters A–F.

  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15

Your Task

  • Ask the user to input a denary number
  • Convert it into a hexadecimal value (maximum 2 digits)
  • Display the result

Hints

  • Use // to find how many 16s fit into the number
  • Use % to find the remainder
  • You will have two values: the 16s column and the 1s column
  • You may need to convert numbers greater than 9 into A–F
  • Consider using a list for conversions
