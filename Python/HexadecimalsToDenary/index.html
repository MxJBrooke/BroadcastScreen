Denary to Hexadecimal Conversion Coding Challenge

Hexadecimal to Denary (2 Digits)

Hexadecimal uses the digits 0–9 and the letters A–F.

  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15

Your Task

  • Ask the user to input a 2-digit hexadecimal number
  • Convert each digit into its denary value
  • Multiply the first digit by 16
  • Add the second digit
  • Display the result

Hints

  • You can access digits using indexing (e.g. number[0], number[1])
  • You will need to convert letters (A–F) into numbers
  • Consider using a list to hold these conversion
