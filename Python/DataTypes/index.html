Data Types

Python Data Types

  • Python uses different data types to store information in variables.
  • int(): Whole numbers (e.g., 1, 42)
  • float(): Decimal numbers (e.g., 3.14, 0.5)
  • str(): Text/strings (e.g., "hello")
  • bool(): True/False values

Python Operators

Python uses different symbols or "operators" to complete maths statements.

Existing Operators: These are calculations you should be familiar with from maths.
  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
New Operators: These are a little more complex and might be new to you.
